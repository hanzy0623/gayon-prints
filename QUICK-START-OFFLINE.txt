## 🚀 RUNNING OFFLINE

### Step 1: Start the Server

Open PowerShell/Terminal and run:

```bash
cd "c:\Users\hannah\OneDrive\Desktop\gayon booth"
python -m http.server 8000 --bind 0.0.0.0
```

You should see:
```
Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...
```

### Step 2a: Access on Same Computer
Visit: `http://localhost:8000`

### Step 2b: Access on Phone/Tablet (Same WiFi)
1. Find your computer IP:
   ```bash
   ipconfig | findstr IPv4
   ```
   (e.g., `192.168.1.37`)

2. On your phone/tablet, visit:
   ```
   http://192.168.1.37:8000
   ```

### Step 3: Use the App
- ✅ Works completely offline after first load
- ✅ No internet needed at event
- ✅ Auto-saves photos
- ✅ Auto-prints (without printer connected)

## 🆘 Troubleshooting

**App says "offline":**
→ Refresh page (Ctrl+R or Cmd+R)

**Camera not working:**
→ Allow camera permission in browser
→ Use HTTPS (http:// with localhost works, might need ngrok for IP)

**Photos not saving:**
→ Check browser storage settings
→ Increase browser cache size
